Constructor 3D interactivo estilo **LEGO** con gráficos realistas usando Three.js. Construye creaciones ilimitadas con bloques de colores vibrantes y controles intuitivos.

## 🎮 Demo en Vivo

**¡Construye ahora!** 👉 [https://jjho05.github.io/BLOCK-BUILDER/app.html](https://jjho05.github.io/BLOCK-BUILDER/app.html)

## ✨ Características

### 🎨 Sistema de Construcción
- **6 Tipos de Piezas**: 1x1, 2x1, 2x2, 2x3, 2x4, 1x4
- **10 Colores LEGO**: Paleta auténtica de colores vibrantes
- **Rotación de Piezas**: Gira bloques 90° con tecla R
- **Modo Borrar**: Elimina bloques individualmente
- **Reinicio Rápido**: Limpia el mundo con un click

### 🖱️ Controles Intuitivos
- **Cámara Orbital**: Rota, zoom y pan con mouse
- **Colocación Precisa**: Sistema de grid automático
- **Vista Previa**: Cursor fantasma muestra dónde se colocará
- **Animaciones**: Bloques aparecen con efecto "pop"

### 🎯 Características Técnicas
- **Geometría Realista**: Bloques con studs (botones) superiores
- **Iluminación Dinámica**: Sombras y reflejos realistas
- **Materiales Phong**: Apariencia de plástico brillante
- **Detección de Colisiones**: Colocación inteligente
- **Efectos de Partículas**: Al borrar bloques

## 🎮 Controles

| Acción | Control |
|--------|---------|
| **Rotar Cámara** | Click Izquierdo + Arrastrar |
| **Zoom** | Rueda del Mouse |
| **Pan (Mover Vista)** | Click Derecho + Arrastrar |
| **Colocar Bloque** | Click Izquierdo |
| **Rotar Pieza** | Tecla R o Botón "Girar 90°" |
| **Modo Borrar** | Botón 🗑️ |
| **Reiniciar** | Botón 🔄 |
